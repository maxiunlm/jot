# JOT (Under Consruction!)

A Javascript Object Toolkits.

## Installation

  `npm install jot`

## Usage

    var jot = require('jot');

    var ioc = jot.getIoC();
    var mapper = jot.getMapper();
    var aop = jot.getAOP();
    var retryManager = aop.getRetryManager();
  
## Tests

  `npm test`
  or
  `yarn test`
